Step 0: Download VNC2FLV using
$ wget https://pypi.python.org/packages/1e/8e/40c71faa24e19dab555eeb25d6c07efbc503e98b0344f0b4c3131f59947f/vnc2flv-20100207.tar.gz

Step 1: Extract the tar
$ tar xvf vnc2flv-20100207.tar.gz

Step 2: Change Dir
$ cd vnc2flv-20100207

Step 3: Install the tar
$ python setup.py install --user

Step 4: change directory to tools
$ cd tools

Step 5: Run the code (assuming you have already started vnc using /opt/TurboVNC/bin/vncserver and obtianed a valid port such as 590(1,2,3,4))
$ python flvrec.py localhost <portno> 
eg. python flvrec.py localhost 5901


At the end of recording press 'Ctrl C' to stop recording and flv file will be generated at launch location.
